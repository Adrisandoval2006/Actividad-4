# **Actividad 4**

Este repositorio contiene el código base del juego *Cannon* tomado de la librería FreeGames

## *Cambios realizados:*
1. Se aumentó la velocidad del proyectil. 
2. Se incrementó la velocidad de los balones (targets), de modo que se desplacen más rápido en pantalla.  
3. Se implementó un sistema de reposición de balones que evita que el juego termine.
4. Los targets que salen de la ventana se reposicionan en el borde derecho con una nueva coordenada y aleatoria, garantizando que el juego sea interminable.
